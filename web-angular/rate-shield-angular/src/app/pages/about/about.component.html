About Rate Shield

Open-source API rate limiting gateway

What is Rate Shield?

A fully configurable rate limiting gateway that applies per-endpoint rules with individual strategies. Integrate it as middleware in any backend — Go, Node.js, Python, and more.

Features

🛠

Customizable Rules

Per-endpoint rules with independent strategies and limits.

⚡

gRPC + HTTP API

Low-latency enforcement via gRPC or standard HTTP.

🔌

Plug-and-Play

Drop-in middleware for Express, Flask, Go Fiber, and more.

📊

Analytics & Audit

Full audit trail and real-time analytics dashboard.

Supported Strategies

Token Bucket

Steady refill rate with burst allowance

Fixed Window

Fixed time window request counting

Sliding Window

Rolling window for smooth rate enforcement
